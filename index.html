¿Quienes somos?

TetuScores nació como un proyecto de fin de curso para la asignatura de Diseño Web de Desarrollo de Aplicaciones Web en el centro Tetuán de las victorias.

Su objetivo estaba claro, crear una página web para la gestión de la liga de fútbol sala que se celebra cada año a principios de curso.

¿Qué ofrecemos?

En TetuScores ofrecemos una plataforma completa para la gestión de ligas de fútbol sala, incluyendo:

  • Calendario de partidos actualizado.
  • Clasificación en tiempo real.
  • Información detallada de equipos y jugadores.
  • Formulario de inscripción para nuevos equipos.
